# Learning Management System

A RESTful API built with Java and Spring Boot for managing learning resources,
featuring an integrated AI assistant to help explain learning content.

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database (in-memory)
- Maven

## Prerequisites
- JDK 21+
- Maven 3.x+

## Getting Started

### Clone the repository
git clone https://github.com/turganaliev/learning-management-system.git

### Run the application
mvn spring-boot:run
